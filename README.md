# Instantgram
A full-stack photo sharing web application built with Flask and AWS cloud services.
## Live Demo
[View Live Page](https://dv19500n.pythonanywhere.com/static/instantgram.html)
## Features
- Upload photos with custom captions
- Automatic unique filename generation
- Responsive grid layout for photo display
- Dynamic content loading without page refresh
- Cloud storage with AWS S3
- Metadata management with AWS DynamoDB
## Tech Stack
- **Backend:** Python, Flask
- **Cloud:** AWS S3, AWS DynamoDB  
- **Frontend:** HTML, CSS3 , JavaScript
- **Deployment:** PythonAnywhere
## API Endpoints
- `POST /uploadpicture` - Upload image with caption
- `GET /listpictures` - Retrieve all images
